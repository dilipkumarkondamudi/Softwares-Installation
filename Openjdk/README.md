# Welcome 2 My World

## Installatins

### 1. Openjdk
### Step-1: 
sudo apt update

### Step-2
java -version

### Step-3
sudo apt install default-jdk

(or)

Example: sudo apt install openjdk-17-jdk

Verification: java -version

### 2. Maven
### Step-1
sudo apt update
### Step-2
sudo apt install default-jdk
### Step-3
sudo apt install maven

Verification: mvn -version
